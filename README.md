This is the Mini Store front Project! a small project where my main goal was to build a small Mini-Storefront using the Next.js App Router. 

How to run

Install dependencies with: npm install
Start the project with: npm run dev
Open http://localhost:3000 in your browser

This projects covers: 
- Components & JSX 
- Lists & Keys 
- Props & Callbacks 
- State & Controlled Inputs 
- Conditional Rendering 
- Effects & Cleanup 
- Lifting State 
- Client vs Server Components 
- API Routes 
- Tailwind UI styling
